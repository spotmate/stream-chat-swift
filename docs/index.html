Stream Chat Swift

Stream Chat SDK consists of three frameworks:

1. Stream Chat Client — the low level library to connect & communicate with StreamChat backend.

2. Stream Chat Core — building on top of Client, it includes models, presenters, and notification manager. Additionally it has RxSwift support for reactive programming.

3. Stream Chat UI — building on top of Core, it's the full SDK library with all UI components. Styling and deep customizations are all supported out of the box.
